# captcha-project

Apliacacion componente captcha

## Ejecucion

Backend `mvn clean install` ->  /target `java -jar backend-biocaptcha-0.0.1-SNAPSHOT.jar`
Frontend `ng serve`

## Tecnologias
 * Java 11
 * Typescript
 * Spring
    - WebFlux/reactor (Rxjava adatpado a spring)
    - OpenApi
    - Swagger
    - Lombok
 * Angular 13
   - Material
   - bootstrap
   - node
