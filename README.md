## Ejecucion

- Backend `mvn clean install` ->  /target `java -jar backend-biocaptcha-0.0.1-SNAPSHOT.jar`
- Frontend `ng serve`

 * Typescript
 * Spring

    - OpenApi
    - Swagger
    - Lombok
 * Angular 13
   - Material

   - node
